# TypeScript typings for Blogger API v3

The Blogger API provides access to posts, comments and pages of a Blogger blog.
For detailed description please check [documentation](https://developers.google.com/blogger/docs/3.0/getting_started).

## Installing

Install typings for Blogger API:

```
npm install @types/gapi.client.blogger-v3 --save-dev
```

## Usage

You need to initialize Google API client in your code:

```typescript
gapi.load('client', () => {
  // now we can use gapi.client
  // ...
});
```

Then load api client wrapper:

```typescript
gapi.client.load(
  'https://blogger.googleapis.com/$discovery/rest?version=v3',
  () => {
    // now we can use:
    // gapi.client.blogger
  },
);
```

```typescript
// Deprecated, use discovery document URL, see https://github.com/google/google-api-javascript-client/blob/master/docs/reference.md#----gapiclientloadname----version----callback--
gapi.client.load('blogger', 'v3', () => {
  // now we can use:
  // gapi.client.blogger
});
```

Don't forget to authenticate your client before sending any request to resources:

```typescript
// declare client_id registered in Google Developers Console
var client_id = '',
  scope = [
    // Manage your Blogger account
    'https://www.googleapis.com/auth/blogger',

    // View your Blogger account
    'https://www.googleapis.com/auth/blogger.readonly',
  ],
  immediate = true;
// ...

gapi.auth.authorize(
  {client_id: client_id, scope: scope, immediate: immediate},
  authResult => {
    if (authResult && !authResult.error) {
      /* handle successful authorization */
    } else {
      /* handle authorization error */
    }
  },
);
```

*/
await gapi.client.blogger.blogs.get({blogId: 'blogId'});
